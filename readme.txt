Плагин оплаты с помощью kaznachey для компонента магазина WooCommerce на базе Wordpress.
Данный плагин интеграции с kaznachey разработан для Wordpress WooCommerce 2.0.x - 2.2.x.
Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "Подключиться"). 
После подписания договора быдут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля

Порядок установки модуля:
1. Выберите меню в консоли WordPress: Плагины / Добавить новый / Загрузить;
2. Выберите файл с архивом (wp-wc-kaznachey.zip) и нажмите "Загрузить";
3. В консоли WordPress выберите пункт меню  Плагины / Установленные, нажмите "Активировать";
4. Перейдите в меню WooCommerce -> Settings (Настройки) -> Payment Gateways (Способы оплаты), Вы увидите доступные способы оплаты. Нажмите на ссылку "kaznachey" для перехода к настройкам плагина;
5. Укажите все необходимые для работы с kaznachey настройки (https://kaznachey.ua);
